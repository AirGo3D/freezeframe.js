Installation

  1. freezeframe.js requires the jQuery framework. Include the latest version of jQuery in your page header before including freezeframe.js.
    http://www.jquery.com

HTML Header

<script type="text/javascript" src="jquery.js"></script>
Note: I recommend using the latest version of jQuery from Google Hosted Libraries
https://developers.google.com/speed/libraries/devguide#jquery

  1. Download the current version of freezeframe.js.

  2. Upload freezeframe.js to your webserver.

  • Optionally, If you are interested in modifying freezeframe in a sandboxed environment, upload the example/ directory to your webserver and modify the contents of example/.

  1. Include the following code in your HTML Header. This will include the script and set it to run on page load. Optionally, you can customize variables that determine how freezeframe functions and looks in the header.

HTML Header

<script type="text/javascript" src="freezeframe.js"></script>
<script type="text/javascript">
freezeframe = new FreezeFrame();
</script>

HTML Header (Optional)

<script type="text/javascript" src="freezeframe.js"></script>
<script type="text/javascript">
freezeframe = new FreezeFrame({
	loading_background_color: "red",
	animation_icon_position: "center center"
});
</script>
Note: The path may need to be modified depending on where you choose to upoad the script.

  1. Add the boolean attribute freezeframe to each image you want modified.

HTML

<img src="whatever.gif" freezeframe />

Styling and Centering the Image

  • To add CSS styling to the image, target the figure tag with the class freezeframe-container wrapped around the image and apply the styles to that element.

CSS

figure.freezeframe-container {
	border-radius: 3px;
}

  • Because freezeframe.js forces the freezeframe-container figure element to be inline, if you want the image to be horizontally centered, apply text-align: center; to the image's parent element.

CSS

div#images-container {
	text-align: center;
}

Use with Infinite-Scroll

  • For use with Infinite-Scroll, follow the steps above and include freezeframe.run() as a new content callback function.
    http://www.infinite-scroll.com

JS

$('#content').infinitescroll({
  option: "value",
  option: "value",
  ...
},function(elements){
	freezeframe.run();
});

Variables

trigger_event

  • Description:

    The input event that will trigger the freezeframe elements to animate. This value is forced to "click" on touch devices if touch_device_support is set to true.

  • Values:

    (string) "hover"
    (string) "click"

  • Default:

    (string) "hover"

trigger_event: "click"

touch_device_support

  • Description:

    Determines freezeframe interaction on touch devices. If set to true, trigger_event on touch devices will be forced to "click", otherwise if set to false, trigger_event on touch devices will be set to the user input trigger_event value.

  • Values:

    (boolean) true
    (boolean) false

  • Default:

    (boolean) true 

touch_device_support: true

animation_play_duration

  • Description:

    The length of time in milliseconds that a GIF will animate if started by a "click" trigger_event. After this period of time the animation will stop.

  • Values:

    (int) [user defined value]
    (undefined)

  • Default:

    (int) 15000 

animation_play_duration: 1500

loading_background_color

  • Description:

    The CSS color of the container shown in place of a GIF while it is processed.

  • Values:

    (string) [user defined value]
    (undefined)

  • Default:

    (string) "#666" 

loading_background_color: "#666"

loading_background_image

  • Description:

    URL to the animated icon shown with the container shown in place of a GIF while it is being processed. Typically an animation to denote loading.

  • Values:

    (string) [user defined image url]
    (boolean) false
    (undefined)

  • Default:

    (string) [base 64 encoded icon] 

loading_background_image: "/images/loading_icon.gif"

loading_background_position

  • Description:

    The CSS position of the animated icon with the container shown in place of a GIF while it is being processed.

  • Values:

    (string) "top left"
    (string) "top center"
    (string) "top right"
    (string) "center left"
    (string) "center center"
    (string) "center right"
    (string) "bottom left"
    (string) "bottom center"
    (string) "bottom right"

  • Default:

    (string) "top left" 

loading_background_position: "top left"

loading_fade_in_speed

  • Description:

    The length of time in milliseconds it will take for the paused GIF to fade in over the loading placeholder.

  • Values:

    (int) [user defined value]
    (undefined)

  • Default:

    (int) 500 

loading_fade_in_speed: 500

animation_icon_image

  • Description:

    URL for the icon layered above the paused GIF. The icon is used to repsent animation or the possiblity of animation (play arrow).

  • Values:

    (string) [user defined image url]
    (boolean) false
    (undefined)

  • Default:

    (string) [base 64 encoded icon] 

animation_icon_image: "/images/animation_icon.gif"

animation_icon_position

  • Description:

    The CSS position of the icon layered above the paused GIF.

  • Values:

    (string) "top left"
    (string) "top center"
    (string) "top right"
    (string) "center left"
    (string) "center center"
    (string) "center right"
    (string) "bottom left"
    (string) "bottom center"
    (string) "bottom right"

  • Default:

    (string) "top left" 

animation_icon_position: "top left"

animation_fade_out_speed

  • Description:

    The length of time in milliseconds it will take for the animted GIF to fade back to the static first frame when the user is no longer triggering animation.

  • Values:

    (int) [user defined value]
    (undefined)

  • Default:

    (int) 300 

animation_fade_out_speed: 300
